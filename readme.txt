Boost Gutenberg editor with new Blocks

== Description ==

This plugin add new blocks to Gutenberg Editor.

It adds following blocks to Gutenberg
* Advanced Divider - It allow you to add Divider with text.
* Dual Heading - Add Dual Color heading. Heading can be split in two parts and each part can be colored separately.
* Progress Bar - Add Multicolor, Animated Progress bar blocks.

Many more blocks coming soon...


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
